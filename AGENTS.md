## Using Agents

### Via Orchestrator
```
/orchestrate Launch a B2B SaaS in France
```
The orchestrator automatically selects the best agents for each task.

### Direct Agent Activation
```
/agent marketing-seo-specialist
```
Activates a specific agent for the current conversation.

### Browse Agents
```
/agents                  — List all divisions
/agents engineering      — List engineering agents
```

### Custom Agents
Place custom agent Markdown files in `./custom-agents/` following the same front-matter format.
